一、基本命令

   1、输出环境变量   # echo $MAVEN_HOME
   2、# scp -r 本地文件 user@ip:/dir
   3、后台进启动
      # nohup java -jar campaign-service-1.0.0-SNAPSHOT.jar 2>&1 &
   4、命令行
      Ctrl+k            删除至行尾
      Ctrl+a            移至行首 
      Ctrl+e            移至行尾 
   5、vim
      Ctrl+f            下一页
      Ctrl+b            上一页   
      :set nu  或者 :set number             显示行号
      :set nonu  或者 :set nonumber         取消显示行号
